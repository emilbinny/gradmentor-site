USAGE:

1. Move this zip into your project folder:
   ~/Documents/gradmentor-site

2. Unzip:
   unzip gradmentor_patch.zip

   chmod +x apply.sh
   ./apply.sh

   git add .
   git commit -m "Applied patch"
   git push
